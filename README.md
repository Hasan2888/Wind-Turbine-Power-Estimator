# Wind Turbine Power Estimator

A Python web app built with Streamlit that estimates wind turbine power output using the physics formula **P = ½ρAv³**.

## Features
- Theoretical vs actual power output comparison (Betz efficiency)
- U.S. regional wind data comparison
- Interactive power curve chart
- NREL U.S. Wind Resource Map
- Adjustable turbine parameters

## Tech Stack
Python, Streamlit, Matplotlib

## How to Run
pip install streamlit matplotlib
streamlit run app.py
